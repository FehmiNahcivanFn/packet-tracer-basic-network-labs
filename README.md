# Packet Tracer Basic Network Labs

Bu repository, Cisco Packet Tracer üzerinde yapılan temel ağ lab çalışmalarını içermektedir.

Amaç; ağ temellerini, IP yapılandırmasını, VLAN yapılandırmasını, switch/router kullanımını ve temel troubleshooting adımlarını lab ortamında uygulayarak öğrenmektir.

## Labs

| Lab | Konu | Durum |
|---|---|---|
| [Basic Static IP Lab](basic-static-ip-lab) | Aynı ağdaki iki bilgisayarın switch üzerinden haberleşmesi | Tamamlandı |
| [VLAN & Inter-VLAN Routing Lab](02-vlan-intervlan-routing-lab) | VLAN, trunk port ve Router-on-a-Stick ile farklı VLAN'lar arası haberleşme | Tamamlandı |

## Kullanılan Araçlar

- Cisco Packet Tracer
- Temel IP yapılandırması
- VLAN
- Trunk Port
- Router-on-a-Stick
- Ping testi
- LAN topolojisi

## Hedef

Bu çalışmalar; Network Support, IT Support ve System Support alanlarında temel ağ bilgisini geliştirmek amacıyla hazırlanmıştır.
